# 📊 Dashboard de E-commerce | Power BI

## 📌 Visão Geral

Este projeto consiste no desenvolvimento de um dashboard completo no Power BI para análise de dados de um e-commerce.

O objetivo é transformar dados brutos em **insights estratégicos**, permitindo a análise de performance de vendas, produtos, categorias, localização geográfica e plataformas.

---

## 🚀 Principais Funcionalidades

O dashboard foi estruturado em 4 páginas principais:

### 🔹 1. Visão Executiva

* Receita Total
* Total de Pedidos
* Quantidade Vendida
* Ticket Médio
* Rating Médio

👉 Foco: visão geral rápida do desempenho do negócio

---

### 🔹 2. Produtos & Categorias

* Top produtos por receita
* Quantidade vendida por produto
* Receita por categoria
* Ticket médio por categoria

👉 Foco: identificar produtos e categorias mais relevantes

---

### 🔹 3. Análise Geográfica

* Receita por cidade
* Ticket médio por cidade
* Quantidade vendida por cidade
* Distribuição geográfica em mapa

👉 Foco: entender o comportamento de vendas por região

---

### 🔹 4. Performance por Plataforma

* Receita por plataforma (Amazon, Jumia, Souq)
* Ticket médio por plataforma
* Comparação entre categorias e avaliações

👉 Foco: avaliar o desempenho dos canais de venda

---

## 🛠️ Tecnologias Utilizadas

* **Power BI**
* **Power Query (ETL)**
* **DAX (Data Analysis Expressions)**

---

## 📂 Estrutura dos Dados

O dataset simula um ambiente real de e-commerce e contém:

* Pedidos
* Produtos
* Categorias
* Preços
* Quantidade vendida
* Avaliações (ratings)
* Cidades
* Plataformas

---

## 🎯 Perguntas de Negócio Respondidas

Este dashboard permite responder perguntas estratégicas como:

### 📈 Vendas & Receita

* Qual é a receita total do negócio?
* Quais produtos geram mais receita?
* Quais categorias possuem maior faturamento?

---

### 🛍️ Produtos

* Quais produtos possuem maior volume de vendas?
* Existe diferença entre produtos mais vendidos e mais rentáveis?
* Quais produtos devem ser priorizados estrategicamente?

---

### 🌍 Geografia

* Quais cidades geram mais receita?
* Onde o ticket médio é mais elevado?
* Existem regiões com potencial de crescimento?

---

### 💻 Plataformas

* Qual plataforma gera mais receita?
* Qual plataforma possui maior ticket médio?
* Existe diferença de comportamento entre plataformas?

---

### ⭐ Experiência do Cliente

* Como está o rating médio geral?
* Existe variação de avaliação entre categorias?
* Quais áreas podem impactar a satisfação do cliente?

---

## 📊 Principais Insights

Com base na análise, é possível:

* Identificar produtos estratégicos para o crescimento do negócio
* Direcionar esforços para regiões mais lucrativas
* Comparar o desempenho entre plataformas de venda
* Entender o comportamento de compra dos clientes
* Apoiar decisões com base em dados
